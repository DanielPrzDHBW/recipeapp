# Recipeapp

App to give you different recipes for each day to cook and a list to shop.

Written in Golang and React.

---
# How to run.

Pull this repository

```git clone https://github.com/DanielPrzDHBW/recipeapp.git```

go into the recipeapp/recipeapp directory

```cd recipeapp/recipeapp```

run the go script

```go run main.go```

You will find the webpage under ```localhost:8080```

## Debugging

In case the webpage is throwing a client error, please clear all cookies of the page and reload.
